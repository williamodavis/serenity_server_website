How To Use Warps, Homes, TPA, and RTP

Warps

Our server has a variety of important locations that can be easily accessed by running the /warp [warpName] command. All warps are listed below.

  • spawn - Teleports you to the world spawn hub and shop board

Homes

Each player on the server can set 3 homes, that they can tp to whenever they want. Commands for them are listed below.

  • /sethome [homeName] - Sets a home at the current location.
  • /delhome [homeName] - Deletes the home with the specified name.
  • /home [homeName] - Teleport to the specified home.

TPA

You can teleport to other players on the server with the /tpa [playerName] command.
After the other player receives the request to teleport, they can type /tpaccept to complete the teleport.

RTP

The rtp command allows players to randomly teleport around the map, within a radius of 3000 blocks around the player, and with a cooldown of 10 minutes. Just run the command /rtp to randomly teleport.
