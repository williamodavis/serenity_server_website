All Plugins

Below is a list of all plugins that involve the player.


  • BetterRTP: Allows the player to RTP, check out this page for info.
  • Gravestones Plus: Creates gravestones when the player dies, so player items do not despawn.
  • GSit: Allows the player to sit on slabs/stairs.
  • HuskClaims: Land Claiming plugin, see this page for more info.
  • mcMMO: Levelling plugin, see this page for more info.
  • SignShop: Allows players to create PtP shops, see this page for more info.
