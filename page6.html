Player to Player Shops

Overview

Sign shops allow players to set up simple, automated trades with other players, like trading one stack of logs for 5 emeralds. Watch the video below for instructions on how to use the sign shops.

Additional Notes

These shops can be set up anywhere, but if you run /warp Diagon_Alley, it will take you to an area where many shops are set up.
